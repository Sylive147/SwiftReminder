# 本地卡片计数器（Cookie 存储）

这是一个本地运行的网页程序，支持：

- 以卡片形式按 `count` 从大到小显示
- 点击 `-` / `+` 调整计数并实时重排动画
- 顶部输入框新增卡片（默认 `count = 1`）
- 所有数据自动保存到浏览器 Cookie

## 数据存储说明

- 每次新增、增减 `count`、删除卡片都会自动写入 Cookie。
- 刷新页面后会自动恢复。
- 若 Cookie 被清空，会自动恢复一组默认示例卡片。

## 默认文件

- 入口页面：`index.html`
- 交互逻辑：`app.js`
- 样式文件：`styles.css`

## 备注

- 由于 Cookie 容量有限，适合轻量数据场景。
- 直接双击 `index.html` 或通过本地服务器打开均可使用。
